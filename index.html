OpenVeganFoodMap

About

This Map is a simple example on how to use the Data from OpenStreetMap (OSM) in custom Map Applications. It shows places that provide Vegan Food, or at least the one that are correctly tagged in OSM. The Sourcecode is available on github. Please use the github issue tracker for bugreports and feature requests.

If you are missing some Places, please add them to OSM.
